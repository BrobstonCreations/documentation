# 2018-2020 Winnebago Revel Front Bed Support

### Tools Needed:

- Impact driver or drill
- 1/4" Hex to 3/8" Socket Adapter
- Hex Socket Set
- Hex Key Set

### Brackets and fasteners included in the kit:
- 1x - Revel Front Bed Support bracket
- 8x - Hex-Drive Flat Head Machine Screws 
- 8x - Threaded Inserts for Wood

![PXL_20251108_164543709.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/PXL_20251108_164543709.jpg)

1. Uninstall the original square tube (used to align the main cushion) and the original bed support.
2. Prepare your impact driver or drill, hex socket adapter, and hex socket.
![1-PXL_20251108_165505383~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/1-PXL_20251108_165505383~2.jpg)
![2-PXL_20251108_165554400~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/2-PXL_20251108_165554400~2.jpg)

3. Align the threaded insert with the first hole and some force, press while pulling the trigger of the impact driver or drill, drive the threaded insert in until it is flush. Install the remaining 7 (2 more for the square tube and 5 more for the support bracket).
![3-PXL_20251108_165630692~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/3-PXL_20251108_165630692~2.jpg)
![4-PXL_20251108_165652275.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/4-PXL_20251108_165652275.jpg)

4. Insert the Hex-Drive Flat Head Machine Screws into the square tube using an hex key to help guide it in.
![5-PXL_20251108_165913446~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/5-PXL_20251108_165913446~2.jpg)
![6-PXL_20251108_165922415~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/6-PXL_20251108_165922415~2.jpg)
![7-PXL_20251108_170013401.MP~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/7-PXL_20251108_170013401.MP~2.jpg)

5. Align the square tube and machine screws with the threaded inserts, tighten (be sure not to over torque), and reinstall the plastic caps.
![8-PXL_20251108_170129695~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/8-PXL_20251108_170129695~2.jpg)
![9-PXL_20251108_170142870~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/9-PXL_20251108_170142870~2.jpg)

6. Insert the Hex-Drive Flat Head Machine Screws into the support bracket, align the bracket and machine screws with the threaded inserts, start the threads by hand, then tighten with a hex key or hex socket; be sure not to over torque.
![10-PXL_20251108_170256785~3.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/10-PXL_20251108_170256785~3.jpg)
![11-PXL_20251108_170422413~2.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/11-PXL_20251108_170422413~2.jpg)

7. You are finished!
![12-PXL_20251108_170756428.jpg](2018-2020-Winnebago-Revel-Front-Bed-Support-images/12-PXL_20251108_170756428.jpg)
